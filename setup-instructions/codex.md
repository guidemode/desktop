# OpenAI Codex Setup

## Installation

OpenAI Codex provides AI-powered code completion and assistance.

### Install Codex

1. **Ensure you have OpenAI API access**
   - Visit [developers.openai.com](https://developers.openai.com/codex/cli/)
2. **Install the Codex CLI** (if available for your platform)
3. **Configure your API key**

### Default Location

Codex stores configuration and sessions in:
- **All platforms**: `~/.codex/`

GuideMode will automatically detect sessions in this directory once Codex is configured and in use.

### Note

Codex availability may vary by region and platform. Check the official OpenAI documentation for the latest information.
