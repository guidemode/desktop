# GitHub Copilot Setup

## Installation

GitHub Copilot CLI provides AI-powered command line assistance.

### Install GitHub Copilot CLI

1. **Ensure you have a GitHub Copilot subscription**
   - Visit [Github Copilot CLI](https://github.com/github/copilot-cli)

2. **Install the CLI**:
   ```bash
   npm install -g @github/copilot
   ```

3. **Authenticate**:
   ```bash
   copilot
   ```

### Default Location

GitHub Copilot stores sessions in:
- **All platforms**: `~/.copilot/`

GuideMode will automatically detect sessions in this directory once you start using Copilot CLI.
