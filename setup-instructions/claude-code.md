# Claude Code Setup

## Installation

Claude Code is Anthropic's official CLI for Claude AI assistance.

### Install Claude Code

1. **Visit the official website**: [claude.com/claude-code](https://claude.com/product/claude-code)
2. **Follow the installation instructions** for your operating system
3. **Authenticate** with your Anthropic account

### Verify Installation

After installation, verify Claude Code is working:

```bash
claude --version
```

### Default Location

Claude Code stores sessions in:
- **macOS/Linux**: `~/.claude/`
- **Windows**: `~/.claude/` (works with WSL/Git Bash)

GuideMode will automatically detect sessions in this directory once Claude Code is installed and you've started using it.
