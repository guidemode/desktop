# OpenCode Setup

## Installation

OpenCode is an open-source AI coding assistant.

### Install OpenCode

1. **Visit the GitHub repository**: [opencode.ai](https://opencode.ai/)
2. **Follow installation instructions** for your platform
3. **Configure** according to the documentation

### Default Location

OpenCode stores sessions in:
- **macOS/Linux**: `~/.local/share/opencode/`
- **Windows**: `%LOCALAPPDATA%/opencode/`

GuideMode will automatically detect sessions in this directory once OpenCode is installed and configured.
